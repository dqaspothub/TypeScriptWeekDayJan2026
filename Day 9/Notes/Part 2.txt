map() ‚Äì ‚ÄúChange EVERY item‚Äù
‚ùì What does map() do?

Takes each element

Performs an operation

Returns a NEW array

Same length as original

üîÅ Mental Model

üëâ ‚ÄúFor every element, give me a modified version.‚Äù

‚úî Use map() when you want to transform data

filter() ‚Äì ‚ÄúKeep ONLY what matches‚Äù
‚ùì What does filter() do?

Checks each element

Returns only elements that satisfy a condition

Output array may be smaller

üîÅ Mental Model

üëâ ‚ÄúPick only the items I want.‚Äù

let evens = numbers.filter(function(num) {
    return num % 2 === 0;
});

console.log(evens);

‚úî Use filter() to remove unwanted data

--------------------------------------------


reduce() ‚Äì ‚ÄúCombine EVERYTHING into ONE value‚Äù
‚ùì What does reduce() do?

Processes all elements

Produces ONE final value

Uses an accumulator

üîÅ Mental Model

üëâ ‚ÄúReduce many values into one.‚Äù

üß™ Example 3: Sum of all numbers
let sum = numbers.reduce(function(total, num) {
    return total + num;
}, 0);

console.log(sum);

‚úî Use reduce() for:

Sum
Average
Counting
Merging data

-----------------------------------

find() ‚Äì ‚ÄúGive me the FIRST match‚Äù
‚ùì What does find() do?

Returns first element that matches condition

Stops searching after first match

Returns value, not array

let result = numbers.find(function(num) {
    return num > 3;
});

console.log(result);

‚úî Use find() when you need one item
===============================================

findIndex() ‚Äì ‚ÄúWhere is the FIRST match?‚Äù
‚ùì What does findIndex() do?

Same as find()

But returns index, not value

üß™ Example 5:
let index = numbers.findIndex(function(num) {
    return num > 3;
});

console.log(index);
================================================
every() ‚Äì ‚ÄúDo ALL elements match?‚Äù
‚ùì What does every() do?

Checks condition for every element

Returns true or false

üß™ Example 6: Are all numbers positive?
let allPositive = numbers.every(function(num) {
    return num > 0;
});

console.log(allPositive);


========================================

some() ‚Äì ‚ÄúDoes AT LEAST ONE match?‚Äù
‚ùì What does some() do?

Checks if any one element matches

Returns true or false

üß™ Example 7: Is there any even number?
let hasEven = numbers.some(function(num) {
    return num % 2 === 0;
});

console.log(hasEven);

===========================================

| Method        | Returns      | Purpose                 |
| ------------- | ------------ | ----------------------- |
| `map()`       | New array    | Transform each element  |
| `filter()`    | New array    | Select elements         |
| `reduce()`    | Single value | Combine data            |
| `find()`      | One value    | First match             |
| `findIndex()` | Index        | Position of first match |
| `every()`     | true/false   | All must match          |
| `some()`      | true/false   | At least one match      |
